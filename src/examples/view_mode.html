View Mode (Read-Only)

View Mode opens the editor in read-only mode. Users can see the design but cannot make changes or save.

Use cases: Order confirmation display, design approval review, customer service viewing, proof verification

Features: No editing tools available, no save/confirm buttons, pure visualization
