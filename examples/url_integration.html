URL Integration (No JavaScript)

URL Integration allows you to open TronDesigner directly via URL parameters without loading any JavaScript.

Use cases: Email links, QR codes, server-side redirects, simple integrations

Benefits: No JavaScript required, works everywhere, easy to generate server-side

Create Mode - Exact Product

Open via URL
Direct link to editor
URL:
{TD_URL_INTEGRATION}/editor?apiKey={TD_API_KEY}&supplierCode=PFC&productCodes=10086021&positionCodes=front&technologyCodes=PAD03

Partial Mode - Model Only

Open via URL
User selects variants
URL:
{TD_URL_INTEGRATION}/editor?apiKey={TD_API_KEY}&supplierCode=PFC&modelCode=100860&dataMode=Partial

Edit Mode - Existing Print Job

URL Format:
{TD_URL_INTEGRATION}/editor?apiKey={TD_API_KEY}&openMode=Edit&printJobGuid=[GUID]

View Mode - Read Only

URL Format:
{TD_URL_INTEGRATION}/editor?apiKey={TD_API_KEY}&openMode=View&printJobGuid=[GUID]

Common URL Parameters

Parameter Description
apiKey Your API key (required)
openMode Create, Edit, or View
dataMode Exact, Partial, or None
supplierCode Supplier identifier
modelCode Product model code
productCode Specific variant code
printPositionCode Print position (e.g., front)
printTechnologyCode Technology (e.g., PAD03)
printJobGuid Existing print job ID
returnUrl URL to redirect after completion
externalUserId Your user identifier
externalOrderId Your order identifier
