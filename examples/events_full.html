Event Handling & Callbacks

TronDesigner provides comprehensive event callbacks to track editor lifecycle and print job changes.

Available Events:

  • printJob.created - New print job created
  • printJob.updated - Existing print job modified
  • printJob.deleted - Print job removed
  • editor.opened - Editor interface opened
  • editor.closed - Editor interface closed

Implementation Code

// Subscribe to events
const td = window.tronDesigner;

td.subscribeDesignerEvent("printJob", "created", (data) => {
    console.log("Print job created:", data.printJobGuid);
});

td.subscribeDesignerEvent("printJob", "updated", (data) => {
    console.log("Print job updated:", data.printJobGuid);
});

td.subscribeDesignerEvent("printJob", "deleted", (data) => {
    console.log("Print job deleted:", data.printJobGuid);
});

td.subscribeDesignerEvent("editor", "opened", (data) => {
    console.log("Editor opened");
});

td.subscribeDesignerEvent("editor", "closed", (data) => {
    console.log("Editor closed");
});
