Session Tracking & Identifiers

Link print jobs to your application's users, orders, and cart items for seamless e-commerce integration.

Available Tracking Fields:

  • externalUserId - Your user identifier (link designs to user accounts)
  • externalOrderId - Your order identifier (connect to specific orders)
  • externalCartItemId - Shopping cart line item tracking
  • customData - Arbitrary JSON metadata for analytics or business logic

Use Cases

E-Commerce Integration

  • Link print jobs to cart items
  • Enable "edit from cart" workflow
  • Track designs through checkout
  • Associate with order after purchase

User Management

  • Display user's past designs
  • Enable design library per user
  • Track user design activity
  • Personalization based on history

Analytics & Reporting

  • Campaign performance tracking
  • A/B test variant tracking
  • Conversion funnel analysis
  • Custom business metrics

Implementation Code

td.openDesigner({
    "openMode": "Create",
    "dataMode": "Exact",
    "printJob": {
        // ... product and printing config ...
        "externalUserId": "user_12345",
        "externalOrderId": "order_67890",
        "externalCartItemId": "item_54321",
        "customData": {
            "campaignId": "summer2024",
            "source": "email",
            "variant": "A"
        }
    }
}, {mode: "dialog"});
