Multiple Print Positions

Configure products with multiple print positions and technologies in a single print job. Users can design for all positions in one session.

Use cases:

  • Front and back printing on apparel
  • Multiple sides on promotional products
  • 360-degree product decoration
  • Multi-location branding (e.g., left chest, sleeve, back)
  • Mixed print technologies (pad print + screen print)

Implementation Examples

1. Using JS Builder API

const td = window.tronDesigner;

td.setExactProduct('PFC', undefined, (b) => {
    b.addVariant('10086021');
});

// Add multiple print positions
td.addExactPrinting('front', 'PAD03');
td.addExactPrinting('back', 'PAD03');
td.addExactPrinting('left', 'SCREEN01');

td.openDesigner(td.argsBuilder.build(), {mode: "dialog"});

2. Using JSON Configuration

td.openDesigner({
    "openMode": "Create",
    "dataMode": "Exact",
    "printJob": {
        "addPrintings": [
            {
                "printPositionCode": "front",
                "printTechnologyCode": "PAD03"
            },
            {
                "printPositionCode": "back",
                "printTechnologyCode": "PAD03"
            }
        ],
        "addProducts": {
            "supplierCode": "PFC",
            "variants": [{"productCode": "10086021"}]
        }
    }
}, {mode: "dialog"});

3. Dynamic from Array

const positions = [
    { position: 'front', technology: 'PAD03' },
    { position: 'back', technology: 'SCREEN01' }
];

positions.forEach(p => {
    td.addExactPrinting(p.position, p.technology);
});

td.openDesigner({
    "openMode": "Create",
    "dataMode": "Exact",
    "printJob": {
        "addPrintings": positions.map(p => ({
            "printPositionCode": p.position,
            "printTechnologyCode": p.technology
        })),
        "addProducts": { /* ... */ }
    }
}, {mode: "dialog"});

Benefits of Multiple Printings

  • Single Session: Design all positions without reopening the editor
  • Consistent Design: Easier to maintain design consistency across positions
  • Better UX: Users don't lose context switching between positions
  • Unified Pricing: Calculate total cost for all positions at once
  • Batch Processing: Generate all print files in one operation
