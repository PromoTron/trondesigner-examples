Editor Customization

Customize the TronDesigner editor appearance, behavior, and functionality to match your brand and workflow requirements.

Customization Options:

  • Appearance: Colors, logo, branding, language
  • Behavior: Skip steps, auto-save, grid display
  • Restrictions: Disable features, limit tools, control uploads
  • Default Design: Pre-populate with text, images, or templates

Customization Reference

1. Appearance Options

"editor": {
    "appearance": {
        "primaryColor": "#ff6600",           // Brand color
        "logoUrl": "https://...",            // Your logo
        "companyName": "My Brand",           // Company name
        "language": "en-US",                 // Language code
        "locale": "en-US",                   // Locale for formatting
        "theme": "light"                     // light or dark
    }
}

2. Behavior Options

"editor": {
    "behavior": {
        "skipProductSelection": true,        // Skip product picker
        "skipTechnologySelection": true,     // Skip tech selection
        "autoSave": true,                    // Enable auto-save
        "showGrid": true,                    // Display design grid
        "snapToGrid": true,                  // Enable snap to grid
        "showRulers": true                   // Show rulers
    }
}

3. Restriction Options

"editor": {
    "restrictions": {
        "disableUpload": false,              // Disable image upload
        "disableText": false,                // Disable text tool
        "disableShapes": false,              // Disable shapes
        "allowedFonts": ["Arial", "..."],    // Restrict font list
        "allowedFileTypes": ["image/png"],   // Limit file types
        "maxFileSize": 5242880,              // Max file size (bytes)
        "maxColors": 3,                      // Max colors in design
        "requireApproval": false             // Require admin approval
    }
}

4. Default Design Options

"editor": {
    "defaultDesign": {
        "addText": {
            "text": "YOUR TEXT",
            "fontFamily": "Arial",
            "fontSize": 24,
            "color": "#000000",
            "x": 50,                         // X position (%)
            "y": 50                          // Y position (%)
        },
        "addImage": {
            "imageUrl": "https://...",
            "x": 50,
            "y": 50,
            "width": 100,
            "height": 100
        }
    }
}
