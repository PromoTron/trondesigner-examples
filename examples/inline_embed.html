Inline Embed Mode

Inline Mode embeds the TronDesigner editor directly into a page element, allowing seamless integration within your page layout.

Use cases: Product detail pages, configurators, seamless design experience, dashboard integrations

Configuration: {mode: "inline", target: "#element-id"}

Control Panel

Editor Container

The editor will appear here when you click one of the buttons above.

Container ID: #editor-container
Size: 100% width × 800px height

Implementation Notes

  • The editor is embedded directly in the #editor-container element
  • No modal overlay or popup window is used
  • You can style the container to match your page design
  • The editor adapts to the container dimensions
  • Multiple inline editors can exist on the same page (different containers)
  • Ideal for product configurators and seamless workflows

Code Example:

tronDesigner.openDesigner({
    "openMode": "Create",
    "dataMode": "Exact",
    "printJob": { /* configuration */ }
}, {
    mode: "inline",
    target: "#editor-container"
});
